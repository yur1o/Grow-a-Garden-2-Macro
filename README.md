# Grow a Garden 2 Macro

Grow a Garden 2 Macro that auto buys seeds and gears. Made for personal use and friends.

## Features

- Toggle individual seeds/gears to include in auto-buy
- Adjustable purchase interval (1–5 minutes)
- Rebindable hotkeys for Start/Stop/Exit
- Live activity log
- Roblox window auto-detection and positioning
- Six color themes (Dracula, Nord, Tokyo Night, Catppuccin, Solarized Light, GitHub Light)

## Download

Grab the latest `.exe` from the [Releases](../../releases) page — no installation required, just run it.

## Usage

1. Launch the app and make sure Roblox is open.
2. Go to the **Shop** tab and pick which seeds or gears you want auto-bought.
3. Set your preferred purchase interval in **Settings**.
4. Press **Start** (or your hotkey) to begin.
